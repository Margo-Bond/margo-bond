# Hi there, I'm Margo 👋

Frontend Developer focused on building clear, reliable, and scalable web interfaces with React, Next.js, and TypeScript.

Most of my recent work has been done in **commercial private repositories**, so this profile does not currently contain public production projects.  
It is intended to reflect my **tech stack, engineering background, and areas of expertise**.

## About Me

I work on frontend applications where not only the UI matters, but also project structure, predictable logic, maintainability, and user experience.

In commercial projects I worked on:

- product interfaces and personal account areas
- authentication and registration flows
- forms with validation and reusable field logic
- API integrations
- responsive and cross-browser layouts
- chat, geolocation, and payment-related flows
- improving code consistency, project structure, and maintainability

I’m especially interested in product development, complex user flows, and frontend solutions that are convenient to scale and support over time.
